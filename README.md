# Neural decay
Цель проекта - реализовать прототип карточной игры, в котором есть главное меню, панель настройки игры, 
панель кастомизации колоды, пара игровых уровней с возможностью победы или проигрыша. 
Механически игра повторяет игру гвинт. Визуально-эстетически игра стремится к гримдарковому киберпанку.

## Реализовано:
1. Drag&Drop система перетаскивания карт
2. Перерасчет очков каждого поля, когда один из игроков положил карту (применяются все ее моментальные эффекты и модификаторы очков)
3. Система состояний матча и раунды (фаза подготовки, ход игрока, смена хода, конец раунда, конец матча)
4. Множество карт, обладающих уникальным набором свойств: 
- Тип карты (обычная, специальная, элита)
- Тип поля и атаки (ближнее-штурм, среднее-огневая поддержка, дальнее-сайбер(cyber))
- Тип способности (вероятность вернуться в руку при проигрыше в раунде, уничтожить самый сильный отряд противника в таком же ряду, умножить значение силы кратно числу таких же карт на поле, и т.д.)
5. Главное меню с анимацией заднего фона
6. Панель настройки игры: разрешение приложения, качество графики, громкость музыки, звуков, мастер
7. Панель кастомизации колоды (Drag&Drop)
8. Подзвучка UI, музыка в меню и в матчах
9. Простой AI для игры против бота
